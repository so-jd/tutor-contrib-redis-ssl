redis-ssl plugin for `Tutor <https://docs.tutor.edly.io>`__
###########################################################

redis-ssl plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/so-jd/tutor-contrib-redis-ssl

Usage
*****

.. code-block:: bash

    tutor plugins enable redis-ssl
